## 🛠️ 使用技術
- React：建構 UI
- React Hook Form：高效能表單管理
- Zod：Schema-based 驗證，整合型別與驗證邏輯
- @hookform/resolvers：整合 React Hook Form 與 Zod
- Vite: 快速啟動與建置
- Tailwind CSS 4

---

## 📦 專案啟動方式

```bash id="4yrkbx"
# 安裝套件
npm install

# 啟動開發
npm run dev
```
